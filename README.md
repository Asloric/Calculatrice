# A simple calulator
The base version is only a python eval() on a text field, with results displayed.
This project was made to replace windows's crappy calculator for something with a real history, and ability to refer to previous results.

## How to use
You can compile/run the calculatrice.py, or launch the .exe (you should not trust anyone's exe, not even mine)
write your math on the left text field with python operators (+, *, -, /, ^ ,...)
you can reference previous results by clicking on it, or typing ans(x) where x is the result number in the list.
